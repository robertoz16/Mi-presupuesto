Mi Presupuesto - Gestor de Ahorro Inteligente

Aplicación integral de gestión financiera diseñada para el seguimiento de gastos y la planificación de metas de ahorro mediante algoritmos dinámicos.

* Características Principales:

1 Calculadora de Ahorro: Genera una tabla personalizada basada en la meta que quieras alcanzar.
2 Control de Gastos: Registro diario para mantener tus finanzas bajo control.
3 Calendario Integrado: Agenda para anotar pendientes y fechas de pago importantes.

* Tecnologías utilizadas:

  1 Visual Studio Code
  2 Html, Css, Javascript 


* Cómo usar la app:

1. Descarga el código.
2. Abre el archivo principal [Mi Presupuesto].
3. ¡Empieza a planificar tu ahorro!
